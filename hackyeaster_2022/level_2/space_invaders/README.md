## Solution

Simply call `pdftotext spaceinvards.pdf` and `cat spaceinvaders.txt` to get:

``` 
he20
22{I nv4d 3rs_
fr0m _sp4 c3!}
```

Then reassemble the flag `he2022{Inv4d3rs_fr0m_sp4c3!}`.
